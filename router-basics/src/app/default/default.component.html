Bienvenue dans les exemples d'utilisation du router

Utiliser les liens ci-dessous pour naviguer en mode Signe Page Application (SPA). Chaque <li> utilise la directive "routerLink" afin de déclencher la navigation au travers du module Router d'Angular plutôt que via le navigateur. Le but de la navigation SPA est de toujours rester sur la même page web

Choisissez les données envoyées :

  • Hello
  • Hello (sans le '/')
  • Une autre URL qui redirige vers Hello
  • URL contenant une variable (urlParam)
  • URL contenant une variable (queryParam)
  • Route avec utilisation de la configuration 'data'
  • Autre route utilisant 'data' avec des valeurs différentes
  • Route avec sous-routes enfants
